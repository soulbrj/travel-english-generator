# 🎬 旅游英语视频课件生成器 - Web版

基于Streamlit的现代化网页应用，一键生成专业的旅游英语学习视频，支持高清下载和多种自定义配置。

## ✨ 核心功能

### 📁 数据管理
- **Excel文件上传**：支持.xlsx/.xls格式，自动解析英语、中文、音标三列
- **示例数据**：内置10个常用旅游英语句子，无需准备数据即可体验
- **在线编辑**：直接在网页中编辑句子内容，实时保存
- **数据验证**：自动检查数据格式和完整性

### ⚙️ 生成设置
- **视频分辨率**：1920x1080（全高清）、1280x720（高清）、854x480（标清）
- **音频模式**：
  - 完整模式（5遍朗读）：英-英-英-中-英，最适合深度学习
  - 标准模式（3遍朗读）：英-中-英，平衡学习效率
  - 快速模式（2遍朗读）：英-中，适合复习巩固
- **字幕样式**：
  - 自定义字体大小（16-60px）
  - 英语/中文/音标独立颜色设置
  - 纯色/渐变/图片多种背景
- **高级设置**：
  - 静默间隔控制（200-2000ms）
  - 语速调节（-50%到+50%）
  - 视频帧率、比特率调节

### 🎬 视频生成
- **实时进度显示**：清晰展示每个生成步骤
- **模拟生成**：演示模式下快速预览效果
- **批量处理**：支持选择特定句子范围生成

### 📥 结果下载
- **高清视频**：MP4格式，支持多种分辨率
- **生成报告**：详细的生成参数和句子列表
- **多格式导出**：支持Excel、JSON、TXT等格式

## 🚀 快速开始

### 方法一：一键运行（推荐）
#### Windows用户
1. 双击运行 `run_streamlit.bat`
2. 等待依赖安装完成
3. 浏览器自动打开 http://localhost:8501

#### macOS/Linux用户
1. 终端中运行：`chmod +x run_streamlit.sh`
2. 运行：`./run_streamlit.sh`
3. 浏览器自动打开 http://localhost:8501

### 方法二：手动安装
```bash
# 1. 克隆或下载项目
git clone <项目地址>
cd travel-english-video-generator

# 2. 安装依赖
pip install -r requirements_streamlit.txt

# 3. 启动应用
streamlit run streamlit_app.py
